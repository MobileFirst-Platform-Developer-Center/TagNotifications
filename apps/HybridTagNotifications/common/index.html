Push Notifications

Broadcast notification

To receive a broadcast notification: right-click the PushAdapter folder in the Studio, select the "sendBroadcastNotification" procedure and provide the notification text in quotation marks, for example: "test notification".

Tag notifications

To receive a notification for a specific tag: first subscribe to a tag below, right-click the PushAdapter folder in the Studio, select the "sendTagNotification" procedure and provide the notification text and tag name in quotation marks, for example: "test notification","sample-tag1" or to notify multiple tags: "test notification","sample-tag1,sample-tag2".
